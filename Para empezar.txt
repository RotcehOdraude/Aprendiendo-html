etiquetas para maquetar para construir un sitio
etiquetas para texto:
<p>: para parrafos
<br/>:para agregar espaciados
<b>,<strong>,'<span>':texto en negritas,strong es una etiqueta mas facil de encontrar por el navegador,la etiqueta span se utiliza en css para darle el estilo bold
<i>:letras en italica
<blockquote>:agregar citas
<cite>:para agregar autor de la cita
<hr/>:es una linea separdora
etiquetas:p,br,b,i,blockquote,hr

"Enlaces"
<a href="UNA_URL">: Sirve para colocar un enlace en la pagina web,href:rutas absolutas(link externo) y rutas relativas(dentro de la ruta raiz)
---enlaces a documentos javascrip---
entlaces a documentos css
<link rel="stylesheet" href="ALGO.css">:etiqueta para agregar css
<script src="ALGO.js">:Enlasa archivos javascript
---enlace a fav icon---
<link rel="shortcut icon" type...>:para agregar el icon de la pestaña
<a href"CORREO_DESTINO" tilte="ALGO">:Activar el correo y enviarlo, no es la forma mas recomendada
etiquetas:a,link.script

"Listas ordenadas y desordenadas"
Listas:Ponerle un contenido para mostrarlo mas facilmente o utilizarlo con css
--lista desordenadas--
<ul>: Para crear lista desordenada
<li>:Elemnto li,contiene nuetro item,
--Listas ordenadas--
<ol>: Para crear lista ordenada
etiquetas:ul,li,ol
--------------------------------------------------------
"Imagenes en html"
<img src="" alt="" title="" width="" height=""/>: atributo src de forma obligatoria, atributo alt sirve para poner un texto en caso de que no se encuentre la imagen en la ruta especificada, el atributo width establece el ancho de la imagen, el atributo height determina el alto de la imagen (Puede escribirse solo un atributo tanto width como height y otro se tomara como predeterminado),el atributo title da una pequeña descripcion de lo que es la imagen
etiquetas:img
-------------------------------------------------------
"Tablas"
Para tabular contenido, mostrarvestadisticas valores, etc.
No se debe utilizar para estructurar un sitio
<table>:Nos sirve para crear la tabla
<tr>:Nos permite crear una fila
<td>:Nos permite crear una columna
etiquetas:table,tr,td
